# qemu-memory-viewer

_A vibe-coded attempt to display RAM of a running QEMU emulation of a 486 with 1 MB of memory running DOS_


## Super-quick Start

Requires: Python 3.10 to 3.13

Install through pip:

```bash
pip install qemu-memory-viewer
```
